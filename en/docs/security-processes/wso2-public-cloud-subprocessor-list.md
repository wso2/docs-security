## WSO2 Subprocessors

When you create a public cloud subscription, your organization administrator user details will be collected and processed by WSO2 for subscription and support services. The following subprocessors would be involved. Please note that your end user or business user data will not be stored within any of the platforms listed below.

Furthermore, when you visit WSO2.com to access documentation or use the Contact Us forms, the following subprocessors get involved.

| Subprocessor | Purpose of processing | Categories of Data Subjects | Location | Transfer Mechanism |
