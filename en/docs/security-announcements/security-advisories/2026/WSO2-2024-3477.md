### AFFECTED PRODUCTS
* WSO2 API Control Plane: 4.6.0, 4.5.0
* WSO2 API Manager: 4.6.0, 4.5.0, 4.4.0, 4.3.0, 4.2.0, 4.1.0, 3.2.1, 3.2.0, 3.1.0
* WSO2 Enterprise Integrator: 6.6.0
* WSO2 Identity Server as Key Manager: 5.10.0
* WSO2 Identity Server: 7.1.0, 5.11.0, 5.10.0
* WSO2 Open Banking AM: 2.0.0
* WSO2 Open Banking IAM: 2.0.0
* WSO2 Traffic Manager: 4.6.0, 4.5.0
* WSO2 Universal Gateway: 4.6.0, 4.5.0


### OVERVIEW
Minor information disclosure vulnerability in HTTP error responses.


### DESCRIPTION
This update collectively delivers essential enhancements to product security by removing unwanted license terms from HTTP error responses to prevent potential disclosure of WSO2 product stack being used.


### IMPACT
This vulnerability only reveals that the WSO2 product stack is utilized within the deployment. It does not expose any other confidential information including version details.


### SOLUTION

#### Community Users (Open Source)
Apply the relevant fixes to your product using the public fix(es) provided below.

* <PR>

If applying the fix or update is not feasible, migrate to the latest unaffected version of the respective WSO2 product(s).


#### Support Subscription Holders

Update your product to the specified update level, or to a higher update level, to mitigate the identified vulnerability.
