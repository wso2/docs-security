### AFFECTED PRODUCTS
* WSO2 API Control Plane: 4.6.0, 4.5.0
* WSO2 API Manager: 4.6.0, 4.5.0, 4.4.0, 4.3.0, 4.2.0, 4.1.0, 3.2.1, 3.2.0, 3.1.0
* WSO2 Identity Server as Key Manager: 5.10.0
* WSO2 Identity Server: 7.2.0, 7.1.0, 7.0.0, 6.1.0, 6.0.0, 5.11.0, 5.10.0
* WSO2 Open Banking AM: 2.0.0
* WSO2 Open Banking IAM: 2.0.0
* WSO2 Traffic Manager: 4.6.0, 4.5.0
* WSO2 Universal Gateway: 4.6.0, 4.5.0


### OVERVIEW
Potential disclosure of sensitive information via Event Publisher warn logs.


### DESCRIPTION
When an Event Publisher output adapter is configured with irrelevant properties, the affected products log the irrelevant properties. This may include user credentials or other confidential information, depending on the property value used.


### IMPACT
A malicious actor with access to the 'wso2carbon' log files could retrieve sensitive information logged by the system, potentially leading to unauthorized access.


### SOLUTION

#### Community Users (Open Source)
Apply the relevant fixes to your product using the public fix(es) provided below.

* [https://github.com/wso2/carbon-analytics-common/pull/882](https://github.com/wso2/carbon-analytics-common/pull/882)

If applying the fix or update is not feasible, migrate to the latest unaffected version of the respective WSO2 product(s).


#### Support Subscription Holders

Update your product to the specified update level, or to a higher update level, to mitigate the identified vulnerability.
