### AFFECTED PRODUCTS
* WSO2 API Manager: 4.2.0, 4.1.0


### OVERVIEW
User Interface Misrepresentation via Swagger UI Try-out Console.


### DESCRIPTION
Due to the insecure usage of the Swagger UI Try-out console in the API Publisher documentation, an external Swagger API definition URL can be loaded, overriding the existing API definitions in the Publisher portal.


### IMPACT
By leveraging this vulnerability, malicious actors could trick users into interacting with overwritten API definitions, potentially exposing sensitive information or causing unintended requests to services.


### SOLUTION

#### Community Users (Open Source)
Migrate to the latest unaffected version of the respective WSO2 product(s).


#### Support Subscription Holders

Update your product to the specified update level, or to a higher update level, to mitigate the identified vulnerability.
