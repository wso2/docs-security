### AFFECTED PRODUCTS
* WSO2 API Control Plane: 4.6.0, 4.5.0
* WSO2 API Manager: 4.6.0, 4.5.0, 4.4.0, 4.3.0, 4.2.0, 4.1.0, 4.0.0
* WSO2 Traffic Manager: 4.6.0, 4.5.0
* WSO2 Universal Gateway: 4.6.0, 4.5.0


### OVERVIEW
SQL injection vulnerability in the Admin REST API.


### DESCRIPTION
The Admin REST API contains a SQL injection vulnerability that allows an authenticated administrator to influence the structure of executed SQL statements, resulting in unintended query manipulation.


### IMPACT
A user with administrator privileges could perform blind SQL injection to infer database information. This may result in the exposure of sensitive data or impact the availability of the database service.


### SOLUTION

#### Community Users (Open Source)
Apply the relevant fixes to your product using the public fix(es) provided below.

* [https://github.com/wso2/carbon-apimgt/pull/13624](https://github.com/wso2/carbon-apimgt/pull/13624)

If applying the fix or update is not feasible, migrate to the latest unaffected version of the respective WSO2 product(s).


#### Support Subscription Holders

Update your product to the specified update level, or to a higher update level, to mitigate the identified vulnerability.
