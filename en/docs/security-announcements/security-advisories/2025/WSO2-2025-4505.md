* WSO2 Enterprise Integrator: 6.6.0
* WSO2 Identity Server: 7.1.0, 5.11.0, 5.10.0
* WSO2 Identity Server as Key Manager: 5.10.0
* WSO2 Open Banking AM: 2.0.0
* WSO2 Open Banking IAM: 2.0.0
* WSO2 Traffic Manager: 4.5.0
* WSO2 Universal Gateway: 4.5.0


### OVERVIEW
An XML External Entity (XXE) vulnerability.


### DESCRIPTION
Due to the improper configuration of XML parser, user-supplied XML is parsed without applying sufficient restrictions, enabling XML External Entity (XXE) resolution.


### IMPACT
A successful XXE attack could allow a remote, unauthenticated attacker to:
* Read sensitive files from the server’s filesystem.
* Perform denial-of-service (DoS) attacks, which can render the affected service unavailable.


### SOLUTION
The recommended solution is to apply the provided update to the affected versions of the products. If there are any instructions given with the update, please make sure those are followed properly.


#### Community Users (Open Source)
Apply the relevant fixes to your product using the public fix(es) provided below.

* [https://github.com/wso2/carbon-mediation/pull/1784](https://github.com/wso2/carbon-mediation/pull/1784)

If applying the fix or update is not feasible, migrate to the latest unaffected version of the respective WSO2 product(s).


#### Support Subscription Holders

Update your product to the specified update level or a higher update level to apply the fix.
