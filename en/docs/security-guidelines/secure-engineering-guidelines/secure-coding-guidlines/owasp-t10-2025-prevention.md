# OWASP Top 10 - 2025 Prevention

<p class="doc-info">Version: 2.0</p>
___

This page maps the OWASP Top 10 - 2025 categories to the WSO2 secure-coding guides. The guides are split by engineering stack:

* [Secure Coding Principles]({{#base_path#}}/security-guidelines/secure-engineering-guidelines/secure-coding-guidlines/principles/) — language-agnostic principles and the public references (OWASP, NIST, RFCs, SLSA) every WSO2 engineer is expected to know. Start here.
* [Java stack guide]({{#base_path#}}/security-guidelines/secure-engineering-guidelines/secure-coding-guidlines/java-stack/) — the established Java/Carbon products.
* [Go stack guide]({{#base_path#}}/security-guidelines/secure-engineering-guidelines/secure-coding-guidlines/go-stack/) — the new Go products.

For each category below, follow the link into the stack guide that applies to the codebase you are working in. Authoritative external references are listed in each guide's section, not repeated here.
